# Titanic passenger survival analysis

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning.

## Dataset
The dataset is taken from Kaggle and contains passenger details such as Survived, Pclass, Sex, Age, SibSp, Parch, Fare and Embarked.

## Steps Involved

### Data Collection
Collected the dataset from Kaggle.

### Data Preprocessing
Handled missing values using SimpleImputer, encoded categorical features using OneHotEncoder and LabelEncoder, and scaled data using StandardScaler.

### Feature Engineering
Improved and modified features to enhance model performance.

### Train-Test Split
Split the dataset into training and testing sets.

### Model Training
Trained multiple models including Logistic Regression, KNN, SVM, and Random Forest.

### Hyperparameter Tuning
Used GridSearchCV to find the best parameters.

## Models Used
- Logistic Regression  
- K-Nearest Neighbors  
- Support Vector Machine  
- Random Forest  
- Voting Classifier  
- Stacking Classifier  

## Libraries Used
- pandas  
- seaborn  
- scikit-learn  

## Evaluation
Model performance is evaluated using accuracy score.

## Conclusion
This project demonstrates an end-to-end machine learning pipeline for a classification problem.
